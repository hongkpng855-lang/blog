## 問題排除指南

### 圖片無法顯示
1. 檢查 `_config.yml` 中的 `baseurl` 設定
2. 確認圖片路徑使用 `{{ site.baseurl }}`
3. 檢查圖片檔案是否存在

### 樣式異常
1. 清除瀏覽器快取
2. 檢查 CSS 檔案是否正確載入
3. 開啟開發者工具查看 404 錯誤

### 部署失敗
1. 檢查 Jekyll 版本相容性
2. 確認 `_config.yml` 語法正確
3. 查看 GitHub Actions 建置日誌
