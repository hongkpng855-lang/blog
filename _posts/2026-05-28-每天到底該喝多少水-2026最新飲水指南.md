「一天要喝八杯水」——這句話你聽過吧？但你有沒有想過，一個 50 公斤的女生和一個 90 公斤的男生，怎麼可能需要喝一樣的水量？

事實上，2026 年台灣國民健康署已經在最新的「國人膳食營養素參考攝取量」草案中，**首度明確列出水分建議攝取量**，讓我們終於有了一個真正科學的飲水標準。今天就來聊聊你真正需要的喝水量，以及那些你經常聽到的喝水迷思。

## 為什麼喝水這麼重要？

先說個你可能不知道的事：人體有 60% 左右是水。水分參與了消化、營養吸收、體溫調節、甚至大腦的思考運作。只要缺水 1-2%，你就會開始感到疲倦、注意力不集中；缺水超過 5%，身體機能就會明顯受影響。

尤其台灣夏天動輒 35 度以上，在戶外走一圈就滿身大汗，水分的流失速度遠比你想像的快。

## 一天到底該喝多少水？最新標準在這

2026 年國民健康署公布的標準，把飲水分成兩種：
- **總水量**：包含所有液體（水、湯、飲料、果汁等）以及食物中所含的水分
- **液體量**：僅計算喝的液體（白開水、湯、飲料等）

### 男性各年齡層建議飲水量

| 年齡 | 總水量 | 液體量 |
|------|--------|--------|
| 0-6 個月 | 700 ml | 700 ml |
| 7-12 個月 | 800 ml | 800 ml |
| 1-3 歲 | 1,300 ml | 1,000 ml |
| 4-6 歲 | 1,700 ml | 1,300 ml |
| 7-9 歲 | 2,000 ml | 1,500 ml |
| 10-12 歲 | 2,300 ml | 1,800 ml |
| 13-15 歲 | 2,600 ml | 2,000 ml |
| 16-18 歲 | 2,600 ml | 2,000 ml |
| **19-75 歲+** | **3,200 ml** | **2,400 ml** |

### 女性各年齡層建議飲水量

| 年齡 | 總水量 | 液體量 |
|------|--------|--------|
| 0-6 個月 | 700 ml | 700 ml |
| 7-12 個月 | 800 ml | 800 ml |
| 1-3 歲 | 1,300 ml | 1,000 ml |
| 4-6 歲 | 1,700 ml | 1,300 ml |
| 7-9 歲 | 1,800 ml | 1,400 ml |

| 16-18 歲 | 2,100 ml | 1,600 ml |
| **19-75 歲+** | **2,700 ml** | **2,100 ml** |

簡單說，成年男性每天至少要喝 **2,400 ml** 的液體，成年女性則是 **2,100 ml**。如果你用 250 ml 的杯子計算，大約是 8-10 杯。

## 更快的方法：用體重算

如果你覺得查表太麻煩，營養師還有一個更簡單的公式：

**體重（公斤） x 30-35 ml = 每日建議飲水量**

舉例：
- 60 公斤的人 → 60 x 30 = 1,800 ml / 天
- 70 公斤的人 → 70 x 33 = 2,310 ml / 天
- 80 公斤的人 → 80 x 35 = 2,800 ml / 天

這個公式的好處是「個人化」，胖瘦高低都能對應到自己的需求。我自己都是用 35 倍去算，因為台灣偏熱，多喝一點沒關係。

## 破解三個常見的喝水迷思

### 迷思一：喝飲料、咖啡不算喝水？

算！而且算得很徹底。所有液體——白開水、無糖茶、咖啡、湯、果汁——都算在「液體量」裡面。不過要注意，含糖飲料喝太多會增加熱量攝取，咖啡因過量也會利尿，**最好的選擇還是白開水**。

### 迷思二：「八杯水」是鐵律？

這是最大的迷思。「八杯水」只是一個便於記憶的口號，真正的需求因人而異。前面提到的年齡對照表就很清楚地告訴你：從嬰兒到老人家，每個階段的需求都不一樣。

### 迷思三：口渴再喝水就好？

口渴其實已經是身體 **輕度缺水** 的訊號了。理想的情況是養成定時喝水的習慣，例如：

- 起床後先喝一杯（300 ml）
- 三餐飯前各一杯
- 下午工作到一半再一杯
- 運動前後補水

## 運動和高溫環境怎麼補水？

夏天跑步、打球、户外工作的朋友要注意：**基本飲水量完全不夠**。

運動營養學建議：
- 運動前 2 小時：喝 500 ml
- 運動中：每 15-20 分鐘喝 150-300 ml
- 運動後：每減輕 1 公斤體重，補充 1,000-1,500 ml

流汗不只是流失水分，還會流失電解質，所以長時間運動可以考慮運動飲料（但要稀釋一半的水）。

## 一句話總結

> **成年男性每天喝 2,400 ml 液體，成年女性 2,100 ml，或用體重 x 35 ml 自己算，夏天運動再多補一些。**

喝水是最簡單、最便宜的養生方式。從今天開始，隨手帶個水壺，設定一下手機鬧鐘提醒，養成定時補水的好習慣，你會發現精神和皮膚狀況都比以前好很多！
