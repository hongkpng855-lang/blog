## 「我知幾時買，但我唔知幾時賣」

你有冇試過呢啲情況？

- 持有一隻股票，升咗 30%，諗住再升啲先賣，結果而家倒蝕
- 跌咗 20%，心諗「一定會反彈」，結果跌多 40%
- 賣咗之後股票繼續升，後悔到想即刻買返
- 明明 set 咗目標價，但到價嗰陣又覺得「仲可以升多啲」

**如果以上任何一項你中咗，你唔孤單。**

我投資咁多年，學到最痛嘅一課就係：**買入決定決定你嘅潛在回報，但賣出決定決定你嘅實際回報。**

2020 年我買咗 Tesla，由 $200 升到 $900，我諗住「一定會上 $1,000 㗎」——結果 $900 跌返去 $300，我嘅利潤由 +350% 變做 +50%。唔係蝕錢，但嗰種「明明賺過咁多」嘅感覺，比蝕錢更難受。

由嗰日開始，我認真研究賣出策略，建立咗一套系統。**呢篇文就係我嘔心瀝血嘅總結。**

## 1. 點解賣出咁難？3 個心理陷阱你要知

### 1.1 處置效應（Disposition Effect）

行為金融學發現：**投資者有強烈傾向「早啲賣出賺錢嘅股票，遲啲賣出蝕錢嘅股票」。**

舉個例：

| 情況 | 正常人會點做 | 理性應該點做 |
|------|------------|------------|
| 股票 A：+20% | 「鎖定利潤！」→ 賣咗 | Check 基本面有冇變 |
| 股票 B：-20% | 「等佢返家鄉先賣」→ 持有 | Check 基本面有冇變 |

**結果：**
- 賣咗 A → A 繼續升多 50% 😭
- 持有 B → B 再跌多 30%，最終忍唔住喺低位賣 😭

呢個就係處置效應——我哋對「蝕錢」嘅痛感係「賺錢」嘅 2 倍，所以我哋寧願 hold 住蝕錢嘅股票，都唔願意承認自己做錯決定。

### 1.2 錨定效應（Anchoring Effect）

你買入嗰刻嘅價格，會成為你腦入面嘅「錨點」：

- 買入價 $100 → 跌到 $80 → 你覺得「蝕緊 $20」
- 但如果你係 $120 先買入，$80 你會覺得「蝕緊 $40」

**同一隻股票，同一價格，因為錨點唔同，你嘅決定完全唔同。**

錨定效應令你抱住「返家鄉」嘅幻想，忽略咗基本面可能已經變壞。

### 1.3 後悔厭惡（Regret Aversion）

「如果我賣咗之後升咗，我會好後悔。所以我唔賣住。」

呢個係最常見嘅唔賣理由。但你要明白：

> **賣出後升咗，代表你做咗正確嘅決定（鎖定利潤/止蝕），只係結果唔完美。但唔賣而跌咗，代表你做咗錯誤嘅決定。**

前者係「好決定，壞結果」；後者係「壞決定，好結果（如果冇跌）」。**長期嚟講，做好決定比好結果重要。**

## 2. 賣出策略嘅 3 大類型

我將賣出策略分為 3 大類，適用於唔同情況：

| 策略類型 | 適用場景 | 核心問題 |
|---------|---------|---------|
| **防守型賣出** | 所有持倉都應該有 | 幾時應該止蝕？ |
| **進攻型賣出** | 主動選股/交易 | 幾時應該獲利？ |
| **系統型賣出** | 被動組合/資產配置 | 幾時應該調整？ |

以下逐個拆解。

---

## 3. 防守型賣出：止蝕策略

**呢個係最基本、最重要嘅賣出策略。無論你做咩類型投資，都一定要有。**

### 3.1 固定百分比止蝕法

最簡單直接：set 一個你接受嘅最大虧損幅度，到價就走。

- **波動大嘅股票（科技股/個股）**：止蝕 set 15-25%
- **波動中嘅股票（大型藍籌）**：止蝕 set 10-15%
- **波動細嘅資產（債券/REITs）**：止蝕 set 5-10%

**實戰建議：** 用「跌穿 xx 日移動平均線」做觸發，唔好只用百分比。例如：

> 「股價跌穿 200 日線，或者由近期高位跌超過 20%，就止蝕。」

### 3.2 移動止蝕法（Trailing Stop）

呢個係我個人最鍾意嘅方法——股價升，止蝕位就升；股價跌，止蝕位唔變。

**例子：**
- 你 set 20% 移動止蝕
- 買入價 $100 → 止蝕位 $80
- 股價升到 $150 → 止蝕位升到 $120（$150 × 80%）
- 股價跌穿 $120 → 自動賣出

**好處：** 唔使估頂，讓利潤奔跑，同時有保護。
**壞處：** 震倉嘅時候可能會被 out。

### 3.3 基本面止蝕法

最理性嘅止蝕方法：**唔係睇股價，係睇基本面有冇變。**

當出現以下信號，就應該賣出，唔理股價：

1. **公司核心業務惡化** — 收入/利潤持續下跌
2. **競爭優勢削弱** — 新競爭者搶走市佔率
3. **管理層出現問題** — 誠信危機/主要人員離職
4. **行業結構性改變** — 技術顛覆/政策改變
5. **估值極端不合理** — PE 升到歷史高位而冇實質支持

> 💡 **最實用組合：** 用「基本面止蝕法」做主框架，用「技術面止蝕法（移動止蝕）」做輔助確認。

## 4. 進攻型賣出：獲利了結策略

呢個比止蝕更難。以下係幾種實戰方法：

### 4.1 目標價賣出法

買入嗰刻就 set 目標價，到價就執行。

**點 set 目標價？**
- **估值目標**：用 PE / PB 歷史區間，set 合理偏高嘅位
- **技術目標**：用阻力位/ Fib retracement
- **回報目標**：唔理估值，set 固定回報（例如 +30%）

**關鍵：** 到價就要執行，唔好改目標。

### 4.2 分批賣出法

最推薦俾一般投資者嘅方法——唔一次過賣晒，分批走：

| 批次 | 觸發條件 | 賣出比例 |
|------|---------|---------|
| 第 1 批 | +20% | 25% |
| 第 2 批 | +40% | 25% |
| 第 3 批 | +60% | 25% |
| 第 4 批 | +80% 或基本面變 | 25% |

**好處：**
- 唔使估頂
- 心理壓力細
- 平均賣出價通常唔錯

**壞處：**
- 永遠唔會賣喺最高位（但呢個係好事嚟）
- 如果得一兩手（例如 ETF），分批冇乜意義

### 4.3 雙目標賣出法

呢個係我而家用緊嘅方法：

> **50% 嘅倉位 set 進取目標，50% 嘅倉位 set 保守目標。**

- 保守目標：+15-20%（一到就走）
- 進取目標：+50-100%（用移動止蝕保護）

**點解有效？**
- 保守部分俾你鎖定利潤，心理安樂
- 進取部分俾你 capture 大升浪
- 唔使兩頭唔到岸

## 5. 系統型賣出：組合管理賣出

### 5.1 再平衡賣出

呢個係被動投資者最主要嘅賣出動作。之前寫過再平衡詳細版，呢度只講重點：

- **每年再平衡 1-2 次**（日曆法）
- **賣出超出目標比例最多嘅資產**
- **買入低於目標比例最多嘅資產**

**再平衡唔係預測市場，係強制你低買高賣。**

### 5.2 人生階段調整

隨住年齡增長，你嘅風險承受能力會下降。建議：

| 年齡 | 股票比例建議 | 賣出動作 |
|------|------------|---------|
| 20-35 | 80-100% | 唔使點賣，持續買入 |
| 35-50 | 60-80% | 每次牛市逐步減持股票 |
| 50-60 | 40-60% | 有計劃地減持，增加債券 |
| 60+ | 20-40% | 維持生活所需，唔好大規模賣出 |

### 5.3 現金需求賣出

當你需要用錢嘅時候，點樣賣出最有效？

**最佳順序：**
1. 先賣出 **偏離目標比例最多** 嘅資產
2. 再賣出 **稅務成本最低** 嘅資產
3. 最後賣出 **表現最差** 嘅資產（除非你認為佢會反彈）

## 6. 賣出決策樹：一張圖搞掂

將以上所有策略組合成一個決策流程：

```
持有投資
│
├─ 你需要現金嗎？
│   ├─ 是 → 按「現金需求賣出法」執行
│   └─ 否
│
├─ 基本面有變壞嗎？
│   ├─ 是 → 立即賣出（唔理價格）
│   └─ 否
│
├─ 股價跌穿止蝕位嗎？
│   ├─ 是 → 執行止蝕
│   └─ 否
│
├─ 股價到達目標價嗎？
│   ├─ 是 → 執行獲利了結（建議分批）
│   └─ 否
│
├─ 組合比例偏離目標嗎？
│   ├─ 是 → 執行再平衡
│   └─ 否
│
└─ 持有，繼續 monitor
```

## 7. 賣出後嘅心理管理

賣出之後，最難嘅部分先開始。

### 7.1 賣出後繼續升？—— FOMO 應對

**心態：** 你鎖定嘅係「你當時認為合理嘅利潤」，唔係「最大可能利潤」。

**行動：** 如果基本面冇變，可以用「原價買返」嘅規則——但只限於股價回落返去你賣出價嘅 5-10% 以內。

### 7.2 止蝕後反彈？—— 唔好 revenge trade

**心態：** 止蝕係保護你嘅資金，唔係預測。10 次止蝕有 3-4 次會反彈，正常。

**行動：** 當係學費。記錄低點解止蝕、點解反彈，下次改善。

### 7.3 建立賣出日記

每次賣出後，記錄：

```
日期：2026-06-11
股票：XXX
賣出原因：基本面惡化（收入連續 2 季下跌）
賣出價：$XX
回報：+15%
後續觀察：_______
學到嘅嘢：_______
```

**每季 review 一次，你會發現自己嘅 pattern。**

## 9. 你嘅賣出策略行動清單

建立你嘅個人賣出系統，只需 4 步：

### Step 1：set 止蝕底線（今天做）
- 每個持倉 set 一個止蝕位（固定 % 或技術位）
- 用移動止蝕保護大賺嘅倉位

### Step 2：set 獲利目標（今天做）
- 每個主動持倉 set 目標價（分批）
- 記錄喺你嘅投資日記

### Step 3：set 再平衡 schedule（本周做）
- 每年 1-2 次再平衡（固定日期）
- 或者 set 偏離閾值（例如超出 5%）

### Step 4：review 系統（每月做）
- 檢查止蝕位有冇需要調整
- Review 賣出日記，搵出自己嘅 pattern

## FAQ

### Q1：ETF 都需要止蝕嗎？
**唔需要**，除非你係短線交易。長線持有 ETF 最好嘅策略係持有 + 再平衡。

### Q2：止蝕 set 幾多比較好？
取決於波動性。大原則：**唔好 set 到一個正常波動就 out 你嘅位**。睇返該資產嘅 historical drawdown 嚟 set。

### Q3：如果賣咗之後即刻升返，應唔應該買返？
如果你嘅賣出原因係「基本面轉差」，唔買。如果只係技術止蝕被震走，可以等回落返去賣出價附近先考慮。

### Q4：定期定額嘅 ETF 點樣賣出？
唔使刻意賣。到你需要用錢嘅時候，用「現金需求賣出法」——先賣偏離比例最多嘅資產。

### Q5：我成日 hold 到賺錢變蝕錢，點算？
呢個係典型嘅處置效應。解決方法：**set 移動止蝕**，俾電腦幫你做決定。

### Q6：一次性賣出定分批賣出好？
一般建議分批。一次性賣出需要你好有信心個 timing 啱——而事實證明，大部分人都冇呢個能力。

### Q7：再平衡嘅賣出同獲利了結有咩分別？
再平衡係系統性嘅（因為比例偏離），獲利了結係主觀判斷嘅（因為到達目標價）。再平衡適合被動投資，獲利了結適合主動投資。
