## 1. 香港IPO生態：點解咁多人抽新股？

香港IPO市場有幾個獨特之處，令佢成為全球散戶最活躍嘅新股市場之一：

### 香港IPO嘅吸引之處

| 特點 | 說明 |
|------|------|
| **回撥機制** | 公開發售部分預留一定比例俾散戶，超額認購愈多，回撥比例愈高（最高50%） |
| **一人一手** | 部分IPO保證每人最少分到一手，散戶入場門檻低 |
| **孖展融資** | 銀行同券商提供高達20-100倍嘅孖展認購，放大回報（同時放大風險） |
| **暗盤交易** | 上市前一晚有暗盤交易，可以提前鎖定利潤或止蝕 |
| **無資本利得稅** | 香港唔徵收股票交易增值稅，賺幾多都唔使交稅 |

### 2026年值得留意嘅大型IPO

根據市場傳聞同已遞交A1申請嘅公司，2026年下半年有機會上市嘅大型IPO包括：

- **字節跳動**（傳聞集資額超過500億港元）
- **SHEIN**（快時尚電商，估值約660億美元）
- **多間AI晶片/生物科技公司**（配合香港推動嘅新特專科技上市機制）

> ⚠️ **注意**：大型IPO唔一定等於好IPO。愈多人抽，分配愈少，潛在回報愈薄。關鍵係計好值博率。

## 2. IPO流程解碼：由招股到上市嘅完整時間線

了解流程係成功抽新股嘅第一步。以下係香港IPO嘅標準時間線：

### 時間線（以T日為上市日）

| 日期 | 事件 | 說明 |
|------|------|------|
| **T-14** | 招股書發布 | 公司向聯交所提交上市文件，公開發售正式開始 |
| **T-14 至 T-3** | 公開認購期 | 一般為3-5日，散戶呢段時間申請認購 |
| **T-2** | 截止認購 | 公開認購結束，定價日（機構部分同日完成） |
| **T-1** | 分配結果公布 | 知道抽唔抽中、分到幾多手 |
| **T-1 下午** | **暗盤交易** | 透過輝立、耀才等券商嘅暗盤系統進行交易 |
| **T日 09:30** | **正式上市** | 港交所首日掛牌交易 |
| **T+2** | 退款 | 未中籤嘅資金退回（孖展客要留意利息計算） |

### 重要節點你要知

**招股期（T-14 至 T-3）**
呢段時間你要：
1. 睇招股書（至少睇summary同風險因素）
2. 決定認購方式（現金/孖展）
3. 比較邊間券商孖展息口最低

**截止日（T-2）**
- 現金認購一般截止前1日截數
- 孖展認購早1-2日截（因為銀行要處理融資）

**分配日（T-1）**
- 早上會收到分配結果（經紀行通知 / 港交所披露易）
- 下午4:15後暗盤開始交易

## 3. Step 1：抽新股前必做嘅分析

唔好見到IPO就衝。抽新股之前，先做呢5個分析：

### 3.1 行業同賽道

- **熱門賽道**：AI、半導體、生物科技、新能源 → 市場願意俾更高估值
- **傳統行業**：地產、零售、製造 → 增長有限，估值受壓
- **當期市場情緒**：牛市IPO普遍表現好，熊市易潛水

### 3.2 估值分析（最重要）

用**市盈率（P/E）**同**市值/收入倍數**同已上市同行比較：

```
估值對比 = IPO發行估值 / 同行平均估值
```

- IPO估值 < 同行平均估值  → 可能有水位
- IPO估值 > 同行平均估值  → 要有強勁增長故事先值博

**實戰例子**：2024年某AI公司上市，發行價對應PS（市銷率）8倍，同行平均PS 12倍 → 估值相對合理

### 3.3 保薦人與承銷團

保薦人係IPO嘅「品質認證」：

- **高盛 / 大摩 / 摩根大通**（國際大行）→ 通常質素較高，但定價偏向機構
- **中金 / 中信 / 海通**（中資券商）→ 散戶認知度高，回撥機會大
- **細行保薦** → 質素參差，小心殼股

**歷史數據**：高盛保薦嘅IPO首日平均回報率約+8%，遠高於市場平均嘅+2%

### 3.4 基石投資者

招股書入面嘅**基石投資者名單**係重要參考指標：

- 有知名基金（如GIC、淡馬錫、BlackRock）做基石 → 正評
- 基石比例 > 50% → 機構信心強，流通量低
- 非公開配售（Placement）比例高 → 小心衝刺上市

### 3.5 孖展認購反應

截止前一日睇各大券商嘅**孖展認購額**：

- 超額倍數 < 5倍 → 冷淡，可能唔夠人抽
- 超額倍數 5-30倍 → 正常
- 超額倍數 > 50倍 → 極度熱爆，分配極少
- 超額倍數 > 100倍 → 回撥機制啟動，分配比例會好低

## 4. Step 2：認購策略——現金 vs 孖展 vs 頂頭槌

### 4.1 現金認購（最穩陣）

**適合**：
- IPO入場費低（< $10,000）
- 唔想俾利息
- 資金有限

**優點**：
- 冇利息成本
- 手續費低（一般$50-100）
- 中籤率同孖展差唔多（一人一手機制）

**缺點**：
- 抽得少，中籤金額有限

### 4.2 孖展認購（槓桿放大）

香港IPO孖展嘅特色係槓桿極高，一般券商提供：

| 券商 | 孖展倍數 | 利息（年化） |
|------|---------|------------|
| 輝立 | 20-100倍 | 1.5%-4% |
| 耀才 | 10-50倍 | 2%-5% |
| 致富 | 10-50倍 | 2.5%-5.5% |
| 銀行（如恒生、中銀） | 5-20倍 | 3%-6% |

**實戰策略**：

**高質IPO（預期首日升10%+）**
→ 借盡孖展，額度愈大愈好
→ 目標係賺差價 — 利息成本佔比低

**中等質素IPO（預期首日升2-5%）**
→ 適量孖展（5-10倍）
→ 利息成本要少過預期升幅

**低質素IPO（唔肯定/行業一般）**
→ 現金認購或唔抽
→ 連孖展都唔值得借

### 4.3 頂頭槌飛（超大額認購）

頂頭槌係公開認購嘅最高額度（一般係公開發售部分嘅49.9%）。

**適合**：
- 極度睇好嘅大IPO
- 資金非常充裕
- 願意承受集中風險

**注意**：
- 頂頭槌有**優先分配機制**，但唔保證賺錢
- 凍結資金量大，利息成本高
- 一單頂頭槌隨時鎖死幾百萬資金成個星期

## 5. Step 3：分配機制解密——點樣提高中籤率？

### 5.1 回撥機制

香港IPO嘅回撥機制係散戶嘅優勢：

| 超額倍數 | 公開發售比例 |
|---------|------------|
| < 15倍 | 10%（基本） |
| 15-50倍 | 30% |
| 50-100倍 | 40% |
| > 100倍 | 50%（上限） |

**意思**：超額倍數愈高，散戶分到嘅貨愈多。

### 5.2 一人一手機制

香港IPO有個不成文規則：**盡量俾每位申請人最少分到一手**。

**實戰策略**：

- **多戶口認購**：用家人、朋友嘅名義開多個戶口，每個都抽一手 → 中籤率遠高過一個人抽多多手
- **點解有效**：一人一手機制下，一個申請抽1手嘅中籤率可能係100%，但抽100手嘅中籤率可能只係10-20%

### 5.3 分配公平性

港交所近年加強咗IPO分配嘅透明度：

- 2023年起所有IPO要公布**完整分配結果**
- 可以喺披露易網站查閱每個申請組別嘅中籤率
- **灰色操作**（頭入頭出、分貨偏幫大戶）大幅減少

## 6. Step 4：暗盤操作——由暗盤睇上市後走势

### 6.1 暗盤係咩？

暗盤係正式上市前一日（T-1 下午4:15-6:30）嘅**場外交易**，主要透過輝立交易場同耀才嘅暗盤系統進行。

### 6.2 暗盤價格點解重要？

暗盤價格反映咗**機構投資者同大戶對IPO嘅真實看法**：

- 暗盤升 → 上市首日大概率升
- 暗盤跌 → 要認真考慮止蝕
- 暗盤冇成交 → 流通量低，上到市都未必活躍

### 6.3 暗盤實戰策略

| 情況 | 策略 |
|------|------|
| **暗盤升 10%+** | 可以考慮食糊，或者留一半搏首日更高 |
| **暗盤升 0-5%** | 視乎預期，如果預期值高可以hold |
| **暗盤跌 0-5%** | 設好止蝕位（建議-3%至-5%） |
| **暗盤跌 5%+** | 強烈考慮暗盤止蝕，首日通常更差 |

> ⚠️ **重要提醒**：暗盤流通量低，價格波動可以好大。唔好用暗盤價格做唯一決定。

## 7. Step 5：賣出時機——首日 vs 持倉策略

### 7.1 首日賣出（最多人用）

**方法**：
- 開市頭30分鐘掛賣盤
- 或者用「競價限價盤」喺集合競價時段賣出

**優點**：
- 鎖定利潤，唔使煩
- 避免持倉過夜風險
- 資金快回籠，準備抽下一隻

**缺點**：
- 可能賣早咗，錯過之後嘅升幅

### 7.2 持倉策略（針對高質IPO）

**適合**：
- 有基石投資者嘅大型IPO
- 行業處於上升週期
- IPO估值相對同行有折讓

**策略**：
- 首日賣一半鎖定利潤
- 另一半持倉睇埋第3-5日
- 設止蝕位（建議-5%）

### 7.3 止蝕紀律（最重要）

**抽新股最常見嘅死法**：跌咗唔肯走，結果由-5%變-50%。

**實戰止蝕規則**：

```
首日跌 > 5% → 一定要走
首日跌 0-5% → 睇暗盤價，暗盤已經跌就即走
持倉跌 > 5% → 唔好猶豫，止蝕
```

## 8. 5大常見IPO錯誤與Fix

### ❌ 錯誤1：乜都抽一餐

**Fix**：建立你嘅IPO篩選清單：
1. ✅ 行業有前景
2. ✅ 估值合理（對比同行）
3. ✅ 有知名基石投資者
4. ✅ 保薦人信譽良好
5. ✅ 市況配合（牛市優先）

符合3項或以上先值得抽。

### ❌ 錯誤2：孖展借到盡

**Fix**：
- 計清楚利息成本佔認購金額嘅百分比
- 預期回報率要大過利息成本2倍先值得借
- 留現金應付退款延遲

### ❌ 錯誤3：上市前唔睇分配結果

**Fix**：
- 分配結果公布後，上港交所披露易睇**中籤率**
- 如果中籤率異常地高（超過50%），代表需求弱，要小心
- 如果只係分到好少（例如抽10手中1手），可以考慮暗盤走

### ❌ 錯誤4：暗盤走勢太好就加注

**Fix**：
- IPO唔係trend trading，暗盤升唔代表一定會繼續升
- 唔好因為暗盤升而喺上市日追貨
- 要買等第3-5日睇清楚走勢先

### ❌ 錯誤5：跌咗當長線投資

**Fix**：
- 抽新股嘅目的是短線套利，唔係價值投資
- IPO首日潛水嘅股票，6個月後有超過60%繼續潛水（統計數據）
- 設定嚴格止蝕，跌穿即走

- **孖展反應**：超額4倍（冷淡）

- **首日收市**：-12%

  - 冇知名基石 → ❌ 應該已經要警覺

  - 暗盤跌3% → ❌ 應該要走
  - 結果硬食-12%

## 10. FAQ

### Q1：抽新股最少要幾多錢？

最低入場費由$2,000-$20,000不等，視乎IPO而定。一般散戶入場門檻係$3,000-$8,000。

### Q2：現金認購同孖展認購，邊個中籤率高？

**對於一人一手機制**，現金認購1手嘅中籤率同孖展認購1手幾乎一樣。但孖展可以認購更多手數，總體中籤機會增大。

### Q3：可以用多個戶口抽同一隻IPO嗎？

可以，只要每個戶口係獨立嘅個人名義。但記住：
- 同一人喺唔同券商用相同身份證號碼申請 → 港交所會detect到，只計算一次
- 用家人/朋友名義開戶 → 每人獨立計算，冇問題

### Q4：新股幾時可以賣？

- **暗盤時段**（T-1 下午4:15-6:30）：提早鎖定利潤或止蝕
- **上市首日**（T日 09:30）：正式交易
- **注意**：部分券商嘅暗盤系統同正式系統唔互通，要分別操作

### Q5：抽中新股唔俾錢會點？

如果中籤後唔俾錢（放棄認購）：
- 會影響你嘅信貸紀錄
- 部分券商會罰款（手續費+利息）
- 嚴重者會被列入黑名單，影響日後認購
- **建議**：認購前預備好資金，唔好搏

### Q6：IPO潛水（跌穿招股價）應該死揸嗎？

**統計上，唔應該。**
- 香港IPO首日潛水嘅股票，3個月後回升嘅比例低於40%
- 死揸嘅期望值係負數
- 嚴格執行-5%止蝕嘅紀律
