## 點解你愈忙，反而做得愈少？

先講一個反直覺嘅概念：**Multi-tasking 係一個謊言。**

神經科學研究已經證實，人腦根本冇辦法同時處理兩樣需要認知資源嘅任務。所謂嘅多工，其實係「快速切換」——每次切換都有切換成本，通常要 10-15 分鐘先可以回復返之前嘅專注水平。

### 多工嘅隱藏成本

| 行為 | 真實影響 |
|------|---------|
| 做緊嘢嗰陣 check email | IQ 暫時下降 10 分（比食咗大麻仲嚴重） |
| 同時間開 5 個 browser tabs | 工作效率下降 40% |
| 每次被打斷後重新專注 | 平均損失 23 分鐘 |
| 一路做嘢一路聽 podcast | 記憶保留率下降 60% |

**真實經歷：**
我以前係 multi-tasking 嘅重度用家——開住 Zoom 做 Excel、一邊寫 code 一邊覆 message。結果點？每日 OT 3 個鐘、成日漏嘢、做到 burnout。直到我開始做深度工作，先發現原來**做得少，真係可以做好啲**。

## 咩係深度工作？唔係叫你收埋自己

深度工作（Deep Work）呢個概念由 Cal Newport 提出，定義好簡單：

> **深度工作 = 喺冇干擾嘅狀態下，進行需要高認知能力嘅工作。**

同之對立嘅係「淺層工作」（Shallow Work）——即係覆 email、開會、碌 Slack 呢啲唔需要用腦嘅任務。

### 深度工作嘅三個層次

**層次 1：環境隔離**（初階）
每日 set 一個「唔受人干擾嘅時段」，最低入場門檻

**層次 2：認知沉浸**（中階）
唔單止環境隔離，仲要有明確目標，進入 flow state

**層次 3：系統化深度**（進階）
整個工作流程同生活節奏都圍繞「深度優先」設計

下面我會由淺入深，逐步教你建立屬於你嘅深度工作系統。

## 第一步：打造「零干擾」工作環境（立即見效）

呢 part 唔使任何技巧，淨係改你嘅工作環境就得。

### 1.1 手機隔離法

最有效嘅方法往往最簡單：**做深度工作嗰陣，將手機放喺第二個房。**

唔係校靜音就算，係要**物理隔離**。研究顯示，就算手機喺你視線範圍內、校咗靜音，佢嘅存在已經會降低你嘅認知能力——因為你個腦嘅一部分會 keep 住留意有冇新通知。

**我嘅做法：**
- 買咗個 $20 嘅計時鎖盒，深度工作時段將手機鎖入去
- 用 Apple Watch 代替手機（只接緊急來電）
- 喺電腦 set Focus Mode，block 咗所有非必要網站

### 1.2 通知管理金字塔

```
頂層：緊急聯絡人（家人、直屬上司）→ 可打斷
中層：團隊協作（Slack、Email）→ 定時 check
底層：社交媒體（IG、FB、WhatsApp group）→ 每日固定時間睇
```

**重要原則：** 預設情況下，**所有通知都應該被 block**，只有你主動允許嘅先可以打斷你。

### 1.3 工作空間設計

枱面愈亂，專注愈難。呢個唔係玄學，係認知心理學：

- **視線範圍內每樣雜物** = 你大腦要處理嘅額外資訊
- **枱面保持「一機＋一本筆記本＋一枝筆」**嘅狀態
- **枱燈用暖光**（2700K-3000K），幫助放鬆入 focus
- **耳機係必備**：降噪耳機 + 白噪音（咖啡廳背景聲最有效）

## 第二步：時間區塊法（Time Blocking）— 真正嘅時間管理核心

與其靠「有時間就做」，不如直接喺 calendar 上面**預約你嘅深度工作時段**。

### 2.1 每日三區塊系統

我嘅每日 schedule 係咁樣分嘅：

```
上午 09:00 - 12:00  🔴 深度區（3 小時）
  - 最重要嘅 1-2 個任務
  - 完全 offline，唔 check message
  - 手機 lock 埋

下午 14:00 - 16:00  🟡 中度區（2 小時）
  - 寫 code / 寫作 / 數據分析
  - 可以有限度 check message（每 30 分鐘一次）

下午 16:00 - 18:00  🟢 淺層區（2 小時）
  - 覆 email、開會、行政工作
  - 所有「唔需要用腦」嘅任務
```

**關鍵：** 深度區係你嘅 peak time，一定要保護。我將所有 meeting 全部推去下晝，朝早 3 個鐘係完全屬於自己嘅。

### 2.2 90 分鐘衝刺法（Pomodoro 升級版）

Pomodoro 嘅 25 分鐘對我嚟講太短——啱啱入到狀態就要停。我用嘅係 **90 分鐘衝刺法**：

1. 揀一個任務（愈具體愈好）
2. 校 90 分鐘倒數
3. 完全唔停、唔 check 任何嘢
4. 到鐘強制休息 20 分鐘
5. 每完成 2 個衝刺，休息一個鐘

**科學根據：** 人嘅注意力自然週期（Ultradian Rhythm）大約係 90 分鐘一個循環。順應呢個節奏，比強迫自己連續做 4 個鐘有效得多。

## 第三步：戰勝拖延嘅實際方法

拖延嘅根源通常唔係懶，而係**任務太大、太模糊、太恐怖**。

### 3.1 5 分鐘啟動法

對住一個大任務覺得無從入手？同自己講：

> 「我只係做 5 分鐘，之後可以停。」

奇蹟係：一旦開始咗，大部分人都會繼續做落去。因為行動本身會產生 momentum。

### 3.2 任務拆解術

將一個大 task 拆到**細到唔可以用「冇時間」做藉口**：

```
❌ 「寫一篇 3000 字文章」
✅ 「開個 blank file → 寫 title → 寫 3 個 bullet point → 每點寫 100 字」
```

每個 sub-task 唔超過 15 分鐘。你做完成一個就剔一格， dopamine 就嚟啦。

### 3.3 拖延日記

每次發現自己拖延，寫低 3 樣嘢：
1. 你諗住要做咩任務？
2. 你實際上做緊咩？
3. 你 feel 到咩情緒？（焦慮？悶？驚做得唔好？）

好快你就會發現 pattern——原來每次要「打電話」或者「面對困難 feedback」嗰陣，你就會走去碌 IG。知道 pattern 之後，你就可以針對性解決。

## 第四步：保護深度時間 — 同同事同老闆溝通嘅技巧

呢 part 可能係最難嘅——點樣同人講「我唔想俾人打擾」，而又唔會得罪人？

### 4.1 set 期望

開一個新 job 嘅時候，同 team 講清楚：

> 「我每日上晝 9 點到 12 點係 focus time，唔會 check message。有 urgent 嘢直接打電話，非 urgent 嘅我會下晝覆返你。」

### 4.2 用 calendar block 晒啲時間

唔好就咁「記住」，要**正式喺 calendar book 咗**。人哋見到你 busy，自然唔會亂咁 invite meeting。

### 4.3 建立「Office Hour」

每星期 set 2-3 個「開放時間」，專俾同事喺嗰段時間問嘢。其他時間嘅非緊急問題，一概唔即時回覆。

## 真實案例：我點樣由 burnout 到每日 3 小時深度工作

2023 年嘅我：每日 OT 到 9 點、週末都要做嘢、成日覺得自己做唔晒、壓力大到瞓唔着。

2024 年開始做深度工作實驗：
- **第一個月**：每日 set 1 小時「冇電話冇 message」時段
- **第三個月**：增加到每日 2 小時
- **第六個月**：建立完整嘅時間區塊系統
- **今日**：每日真正深度工作 3 小時，產出比以前 OT 4 個鐘仲要多

**數據對比：**

| 項目 | 之前（無系統） | 之後（深度工作） |
|------|-------------|---------------|
| 每日實際工作時間 | 10-12 小時 | 6-7 小時 |
| 真正有生產力時間 | ~2 小時 | ~4.5 小時 |
| 每週完成 task 數 | 15-18 個 | 22-28 個 |
| 睡眠質素 | 差（成日諗嘢） | 好（收工就放低） |
| 壓力水平 | 8/10 | 4/10 |

## 常見問題 FAQ

### Q1：我做創意工作，深度工作會唔會扼殺靈感？
唔會。深度工作係幫你**執行**靈感，而唔係扼殺靈感。靈感可以喺沖涼、行路嘅時候出現，但將靈感變成成品，就需要深度工作。

### Q2：我份工成日要覆客，點樣深度工作？
試用「批次處理法」：set 每日 3 個時段覆客（例如 10am、2pm、4pm），其餘時間唔 check。客通常會接受——只要你 reply 得快就得。

### Q3：WFH 好難專注，有咩計？
- 建立「開工儀式」：換衫、沖杯咖啡、坐定某個位
- 用「虛擬 coworking」：同朋友開住 Zoom 自己做自己嘢
- 如果屋企真係太亂，去附近嘅自修室或者咖啡店

### Q4：我試過做深度工作，但堅持唔到一個禮拜...
正常！由每日 30 分鐘開始。**唔好一步就想做到 3 小時。** 由細開始，慢慢建立習慣，先係持久嘅方法。

## 總結：今日就可以開始嘅 3 個行動

唔使睇完整篇文章先開始。今日就可以做呢 3 件事：

1. **校 30 分鐘倒數**，完全唔掂電話，淨係做一件正經事
2. **刪除 3 個最冇用嘅 notification**（我保證你唔會 miss 任何重要嘢）
3. **聽日 calendar 預留朝早 1 小時**，寫住「深度工作，請勿打擾」

記住：**你做唔晒所有嘢，但你可以做好最重要嘅嘢。** 喺呢個不停搶你注意嘅世界入面，專注力就係你最珍貴嘅資源。

保護好佢，就好似保護你銀包入面嘅錢一樣。
